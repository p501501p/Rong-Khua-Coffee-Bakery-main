# ☕ Rong Khua Coffee & Bakery

## 📖 รายละเอียดโครงการ

Rong Khua Coffee & Bakery เป็นเว็บไซต์สำหรับร้านกาแฟและเบเกอรี่ที่พัฒนาขึ้นเพื่ออำนวยความสะดวกให้กับลูกค้าในการเลือกซื้อสินค้าและสั่งซื้อผ่านระบบออนไลน์ รวมถึงช่วยให้ผู้ดูแลระบบสามารถจัดการข้อมูลสินค้าและคำสั่งซื้อได้อย่างมีประสิทธิภาพ

โครงการนี้ถูกพัฒนาขึ้นเพื่อศึกษาการพัฒนาเว็บแอปพลิเคชัน (Web Application) โดยใช้ React ในการพัฒนา Front-end เพื่อสร้าง Single Page Application (SPA) ที่มีการแสดงผลรวดเร็วและตอบสนองต่อผู้ใช้งาน

---

## 🎯 วัตถุประสงค์ของโครงการ

- พัฒนาเว็บไซต์สำหรับร้านกาแฟและเบเกอรี่ให้สามารถใช้งานได้จริง
- ศึกษาการพัฒนา Single Page Application (SPA) ด้วย React
- เรียนรู้การออกแบบ Component และการจัดการ State ด้วย React Hooks
- ส่งเสริมการทำงานร่วมกันเป็นทีมโดยใช้ Git และ GitHub

## 🚀 วิธีติดตั้ง

1. Clone โปรเจกต์

```bash
git clone https://github.com/p501501p/Rong-Khua-Coffee-Bakery-main.git
```

2. เข้าโฟลเดอร์โปรเจกต์

```bash
cd Rong-Khua-Coffee-Bakery-main
```

3. ติดตั้ง Package

```bash
npm install
```

4. เริ่มต้นโปรแกรม

```bash
npm run dev
```
